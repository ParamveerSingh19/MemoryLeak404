# 🧠 MemoryLeak404

> A cyberpunk-themed memory matching game where **your brain is the garbage collector.** Match the corrupted emoji pairs before the system leaks.

A fast-paced and visually engaging **Memory Match** game built with **React, Tailwind CSS, and pure CSS animations**. Challenge your memory across multiple difficulty levels while enjoying a neon cyberpunk aesthetic with smooth animations and responsive gameplay.

Works seamlessly on **Desktop, Tablet, and Mobile**.

---

## 🚀 Features

- 🎮 Smooth **Welcome → Game → Victory** flow
- 🧩 **5 progressively challenging levels**
- 🃏 Beautiful 3D card flip animations
- ⚡ Cyberpunk-inspired UI with glitch effects and neon styling
- ⏱️ Real-time timer and move counter
- 📊 Live progress bar to track your progress
- ⏸️ Pause and resume gameplay anytime (or press **Esc**)
- ⭐ Performance-based star rating system
- 🏆 Best time saved for every level using **localStorage**
- 🔔 Match and miss feedback with animated neon toast notifications
- 📱 Fully responsive design for desktop, tablet, and mobile devices
- 💾 Completely client-side with persistent local storage
- 🌐 Works offline after the first load

## 🎮 How to Play

1. Select a difficulty level from the welcome screen.
2. Press **Start** to begin.
3. Flip two cards by clicking or tapping them.
4. Match identical emoji pairs to keep them revealed.
5. Incorrect matches flip back after a short delay.
6. Match every pair to complete the level.
7. Try to beat your best completion time and earn a higher star rating.

## 💡 Gameplay Tips

- Memorize card positions before flipping more cards.
- Fewer moves result in a higher star rating.
- Keep an eye on the timer to beat your personal best.
- Use the pause feature whenever you need a break.
- Practice easier levels before attempting **TERA_404**.

## 🚀 Running Locally

1. **Clone the repository**

```bash
git clone https://github.com/ParamveerSingh19/MemoryLeak404.git
```

2. **Navigate into the project folder**

```bash
cd MemoryLeak404
```

3. **Install dependencies**

```bash
npm install
```

4. **Start the development server**

```bash
npm start
```

Then open:

```
http://localhost:3000
```
